# Frontend - Biblioteka

React приложение для управления библиотекой книг.

## Быстрый старт

1. Установить зависимости:
```bash
npm install
```

2. Запустить development server:
```bash
npm start
```

Приложение откроется на http://localhost:3000

## Структура проекта

```
src/
├── components/        # React компоненты
│   ├── Header.js      # Шапка сайта (с цитатой, меню пользователя, выбор библиотек)
│   ├── Sidebar.js     # Боковая панель (категории с счетчиками, хэштеги)
│   ├── BookCard.js    # Карточка книги
│   ├── BookGrid.js    # Сетка карточек
│   ├── Filters.js     # Фильтры книг
│   ├── LibrarySelector.js # Выбор библиотек (dropdown с чекбоксами)
│   └── PrivateRoute.js # Защищенный маршрут
├── pages/            # Страницы
│   ├── LoginPage.js   # Страница авторизации
│   ├── MainPage.js    # Главная страница (список книг с фильтрами)
│   └── UserPage.js    # Страница пользователя (профиль, управление библиотеками)
├── services/          # API сервисы
│   └── api.js        # Axios клиент и API методы
└── utils/            # Утилиты
    └── auth.js       # Утилиты для работы с токенами
```

## Основные функции

### Главная страница (MainPage)
- Отображение книг с фильтрацией по категориям, хэштегам и библиотекам
- Динамические счетчики книг в категориях с учетом выбранных библиотек
- Если не выбрана ни одна библиотека - книги не отображаются
- Поиск и фильтры по статусу, отзывам, электронным версиям

### Страница пользователя (UserPage)
- Редактирование профиля (ФИО, фото, описание)
- Управление библиотеками (создание, удаление)
- Просмотр количества книг в каждой библиотеке

### Компонент LibrarySelector
- Отображается в шапке сайта
- Позволяет выбрать одну или несколько библиотек
- Свои библиотеки отображаются первыми и выделены жирным
- Кнопка "Выбрать все" для выбора всех библиотек
- Выбранные библиотеки сохраняются в localStorage

### Компонент Sidebar
- Отображение иерархии категорий с счетчиками книг
- Динамические хэштеги с визуализацией частоты (размер шрифта от 8px до 20px)
- Счетчики категорий обновляются мгновенно при изменении библиотек (оптимизировано на бэкенде)
- Подсчет книг происходит на сервере за один запрос (~200-300x быстрее)
